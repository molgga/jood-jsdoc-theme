# @jood/jsdoc-theme

A clean, responsive documentation template theme for JSDoc 3, based on LatoDoc

## Uses

```
$ npm install @jood/jsdoc-theme --save-dev
```

`.jsdoc.json`

```json
{
  "opts": {
    "template": "node_modules/@jood/jsdoc-theme"
  }
}
```
